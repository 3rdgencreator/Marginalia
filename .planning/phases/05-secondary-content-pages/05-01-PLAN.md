| content/*.yaml → reader.singletons | Seed YAML files are read at build/render time; malformed YAML causes build errors |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-01-01 | Tampering | keystatic.config.ts image path config | mitigate | `directory` must NOT have leading slash (`public/images/about` not `/images/about`) and `publicPath` must have both slashes (`/images/about/`); enforced in acceptance_criteria |
| T-05-01-02 | Denial of Service | content/home.yaml seed file | mitigate | Seed file must exist with valid YAML before any page calls `.read()`; absence causes null return but not crash (using `.read()` not `.readOrThrow()`) |
| T-05-01-03 | Tampering | content/about.yaml `body: []` | accept | Body field is an empty document array — correctly typed; DocumentRenderer handles empty array gracefully |
</threat_model>
